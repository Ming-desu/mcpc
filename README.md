# mcpc

Queueing System for Patients built with Nodejs and Express

## Pre-requisites

- Nodejs >= v14.17.0

## Getting Started

1. Clone this repository

```bash
  git clone https://github.com/Ming-desu/mcpc.git
```

2. Install required dependencies

```bash
  cd mcpc && npm install
```

3. Run development server

```bash
  npm run dev
```
